## 📦 申报材料清单

### 基础信息
- [ ] 身份证/护照信息
- [ ] 最高学历证书
- [ ] 个人简历
- [ ] 证件照（白底职业照）

### 项目材料
- [ ] 公司营业执照（若已注册）
- [ ] 商业计划书（精简版1-2页）
- [ ] 核心数据：用户数、GMV、融资阶段、营收
- [ ] 媒体报道截图
- [ ] 获奖证书/资质认证

### 推荐信
- [ ] 至少1封知名投资人或机构背书
- [ ] 建议搭配1封行业专家/学者推荐

### Essay类
- [ ] 创始人故事（约500字，要有记忆点）
- [ ] 项目创新性描述（约300字）
- [ ] 社会价值阐述（约200字）
